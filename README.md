# Healthcare-Data-Analysis (Interactive Dashboard creation using Ms Excel)
## Project Objective
The goal of this project is to explore patient activity, treatment costs, recovery outcomes, and readmission risk to generate useful healthcare insights that could support NHS-style operational reporting and decision-making.
## Dataset used
<a href="https://github.com/oreofe2026/healthcare_data_analysis_dashboard/blob/main/healthcare_patient_analytics_seaborn.csv">Healthcare Data Set</a>
## Questions (KPIs)
1. Which region has the highest average treatment cost?
2. What is the average length of stay by department?
3. Which treatment type has the highest average cost?
5. What is the average recovery score by treatment type?
